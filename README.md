#Collapsible.css

Pure CSS collapsibles elements

##Usage

```html
<div data-collapsible>

    <input id="collapsed-stuff" type="checkbox">
    <label for="collapsed-stuff" onclick>Click me</label>

    <div class="collapsible-content">
        I'm a collapsed text
    </div>

</div>
```
